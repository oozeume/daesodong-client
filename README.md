# 대소동


동물병원 커뮤니티 서비스. 대형 커뮤니티가 아직 활성화되지 않은 특수동물을 대상으로 하며 활성화를 목표로 합니다. 

<br />

## Feature
- 지도 웹뷰, 지도 검색
- 병원 리뷰 작성, 수정, 삭제, 공유, 별점
- 마이페이지
- 커뮤니티
- 콘텐츠
